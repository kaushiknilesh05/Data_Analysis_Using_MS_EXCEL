# Data_Analysis_Using_MS_EXCEL
This project presents a comprehensive analysis of pizza sales data using Microsoft Excel. The dashboard provides actionable business insights through interactive visualizations, KPI metrics, and sales trend analysis.<br>
The objective of this project is to demonstrate data analysis, dashboard design, data visualization, and business intelligence skills using Excel.
